# agent-memory-session-core

## 项目目标
搭建Agent短期会话记忆核心模块，实现对话历史存储、上下文管理、超长上下文自动裁剪，解决多轮对话上下文冗余问题。

## 建设内容

1. 会话级历史消息存储与读取能力
2. 上下文滑动窗口截断算法
3. 消息角色分类管理（用户/助手/工具）
4. 会话状态持久化与恢复机制
5. 记忆检索与相关性匹配基础能力

## 核心功能点

- 多轮对话历史自动维护
- 智能上下文长度管理
- 会话隔离，多用户数据不串扰
- 支持本地/内存双模式存储
- 记忆检索与上下文注入

独立开发者自研，对标 OpenClaw 多Agent调度架构
